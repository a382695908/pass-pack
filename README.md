# PASS-PACK

> 解决密码总是忘记的问题

将密码存储在本地用户目录下，使用AES对称加密算法，使用者只需记住加密密钥，便可将密码解密出来

### 安装

```
npm install pass-packet -g
```

### 使用

在shell或cmd中输入`pass-packet`
